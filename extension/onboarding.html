به PechPech خوش آمدید 👋

افزونه‌ای که جلسات آنلاین شما را ضبط می‌کند و بدون هیچ یادداشتی، صورت‌جلسه فارسی آماده تحویل می‌دهد.

🎙
ضبط هوشمند جلسه
صدای تب مرورگر (Google Meet، Teams، Zoom Web) و میکروفون شما را همزمان ضبط و ترکیب می‌کند. هیچ نرم‌افزار جانبی لازم نیست.
📝
تبدیل گفتار به متن با Whisper
صدای ضبط‌شده به سرویس Whisper ارسال می‌شود و به متن دقیق فارسی تبدیل می‌گردد — با پشتیبانی از لهجه و اصطلاحات فنی.
📄
صورت‌جلسه ساختارمند
یک مدل زبانی (Claude، GPT یا هر LLM دیگری) متن را تحلیل کرده و خلاصه، تصمیمات، و اقدامات جلسه را به فارسی می‌نویسد.

۱
افزونه Chrome
صدای جلسه را ضبط و به سرور محلی ارسال می‌کند. همه پردازش‌ها روی سیستم خودتان انجام می‌شود — هیچ داده‌ای به سرور خارجی نمی‌رود.
۲
سرور محلی (Node.js)
فایل صوتی را دریافت، پاکسازی و به Whisper ارسال می‌کند. متن دریافتی را به LLM می‌دهد و صورت‌جلسه نهایی را ذخیره می‌کند.
۳
Whisper STT
یک سرویس محلی یا ابری سازگار با OpenAI API که صدا را به متن فارسی تبدیل می‌کند. می‌توانید از Groq، OpenAI، یا whisper.cpp محلی استفاده کنید.
۴
مدل زبانی (LLM)
Claude Code (پیشنهادی)، یا هر API سازگار با OpenAI، یا هر CLI سفارشی. متن جلسه را تحلیل و صورت‌جلسه می‌نویسد.

راه‌اندازی اولیه

اگر هنوز سرور را نصب نکرده‌اید، این مراحل را طی کنید. اگر قبلاً install.sh را اجرا کرده‌اید، از این مرحله رد شوید.

۱
پیش‌نیازها را نصب کنید
Node.js 18+، Docker Desktop، و Claude CLI (یا هر LLM دیگری) روی سیستم داشته باشید.
۲
اسکریپت نصب را اجرا کنید
در پوشه اصلی PechPech:
chmod +x install.sh && ./install.sh
این اسکریپت Whisper و تنظیمات LLM را پیکربندی می‌کند.
۳
سرور را اجرا کنید
node src/launcher/index.js
سرور روی پورت ۳۴۵۶ بالا می‌آید. این پنجره را باز نگه دارید.
۴
افزونه را در Chrome بارگذاری کنید
به chrome://extensions بروید، Developer mode را روشن کنید، روی «Load unpacked» کلیک کنید و پوشه src/extension را انتخاب کنید.

آماده‌اید!

PechPech نصب شده و منتظر اولین جلسه شماست.

🎙 در جلسه بعدی، تب Google Meet یا Teams را باز کنید، روی آیکون PechPech در نوار Chrome کلیک کنید و شروع ضبط را بزنید.
⏹ در پایان جلسه توقف و ذخیره را بزنید. فایل صوتی به سرور ارسال می‌شود و پردازش شروع می‌شود.
📄 چند دقیقه صبر کنید. صورت‌جلسه نهایی (خلاصه + تصمیمات + اقدامات) در پاپ‌آپ افزونه ظاهر می‌شود.
✏️ اگر متن جلسه نیاز به اصلاح دارد، دکمه اصلاح رونوشت را بزنید تا LLM متن را بازبینی کند.
