به PechPech خوش آمدید

ابزاری برای تولید خودکار صورت‌جلسه — بدون هیچ یادداشت‌برداری در حین جلسه. فقط ضبط کنید، بقیه کار را ما انجام می‌دهیم.

🎙
ضبط صدای جلسه صدای تب مرورگر (Google Meet، Teams، Zoom Web) و میکروفون شما را همزمان ضبط می‌کند.
📝
تبدیل به متن با هوش مصنوعی صدا از طریق سرویس Whisper به متن تبدیل می‌شود. از شناسایی گویندگان مختلف پشتیبانی می‌کند.
📄
تولید صورت‌جلسه فارسی خلاصه، تصمیمات، و اقدامات جلسه را به صورت ساختارمند و فارسی تولید می‌کند.

نصب افزونه Chrome

افزونه PechPech باید به صورت دستی در Chrome بارگذاری شود. این فقط یک بار لازم است.

۱
صفحه افزونه‌های Chrome را باز کنید در نوار آدرس Chrome تایپ کنید:
chrome://extensions
۲
Developer mode را فعال کنید کلید «Developer mode» را در بالا سمت راست صفحه روشن کنید.
۳
روی «Load unpacked» کلیک کنید دکمه‌ای در بالا سمت چپ ظاهر می‌شود.
۴
پوشه extension را انتخاب کنید پوشه extension/ داخل دایرکتوری PechPech را انتخاب کنید.
۵
آیکون افزونه در نوار ابزار ظاهر می‌شود اگر نمایش نشد، روی آیکون پازل در نوار Chrome کلیک کرده و PechPech را پین کنید.

تنظیمات

روی آیکون افزونه در Chrome کلیک کنید، سپس آیکون تنظیمات (⚙) را بزنید. این فیلدها را پر کنید:

فیلد توضیح
STT URL آدرس سرور Whisper. مثال: http://localhost:8080/v1
STT API Key کلید API سرویس Whisper. برای سرورهای محلی خالی بگذارید.
STT Model نام مدل. برای OpenAI: whisper-1
LLM CLI نوع مدل زبانی. پیشنهاد: claude
Helper Port پورت سرور محلی. پیش‌فرض: 3456

آماده‌اید!

PechPech در حال اجرا است و آماده ضبط اولین جلسه شماست.

🎙 برای ضبط جلسه روی آیکون PechPech در نوار ابزار Chrome کلیک کنید.
⏹ در پایان جلسه «توقف و ذخیره» را بزنید، سپس روی «پردازش» کلیک کنید.
🖥 آیکون PechPech در نوار وظیفه سیستم شما همیشه در دسترس است. برای شروع خودکار با ورود به سیستم، «Launch on Login» را از منوی آیکون فعال کنید.
